# JSON Beautifier

Projeto simples para formatação de JSON, deixando o conteúdo legível com indentação adequada.

## O que faz

- Recebe JSON bruto ou arquivo JSON
- Aplica formatação (beautify)
- Produz saída JSON com indentação de 2 espaços

## Como rodar

1. Instale dependências:

   ```bash
   npm install
   ```

2. Execute o projeto:

   ```bash
   npm start
   ```

3. Se a aplicação for um script Node.js, execute diretamente:

   ```bash
   node index.js
   ```

## Observações

- Verifique o arquivo principal do projeto para ajustar o comando direto se o nome não for `index.js`.
- O comando `npm start` deve estar definido no `package.json` para executar o script corretamente.
